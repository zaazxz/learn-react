----- React -----
[ Components ]
* Components dalam react adalah sebuah konsep pemecahan elemen menjadi bagian-bagian 
  kecil yang independent (berdiri sendiri) dengan tujuan membuat element tersebut reusable 
  (dapat digunaan kembali).
* Component di react terbagi menjadi 2 yakni Class Component dan Function Component.
* Yang membedakan keduanya adalah cara penulisan, versioning react, dan tujuan penggunaan.
* Nama lainnya adalah Statefull Component (Class) dan Stateless Component (Function).
* Aturan penulisan sebuah component salah satunya adalah menggunakan Pascal Case.

[ Props ]
* Setiap components, pasti memiliki props.
* Props adalah sebuah object yang berisi data-data yang dibutuhkan oleh component.

[ Konsep Atomic Design ]
* Atomic Design adalah konsep yang dibuat oleh Brad Frost untuk pemecahan component.
* Atomic Design adalah sebuah metodologi yang terdiri dari 5 tahapan berbeda yang dimana 5 tahapan nya
  saling terhubung dan bekerjasama untuk membuat hierarki system dalam UI sebuah aplikasi.
* https://atomicdesign.bradfrost.com/chapter-2/

[ State ]
* State adalah sebuah memori yang dimiliki oleh sebuah component.
* Sebuah component dapat menyimpan sebuah data yang dapat dimanipulasi. 
* State pada Statefull Component dan Stateless Component berbeda.
* Untuk menggunakan state sebaiknya menggunakan statefull component (Class Component).
* Tetapi stateless component (Function Component) pun bisa menggunakan state dengan bantuan useState().

[ React Hooks ]
* Pada versi React 16.8.0, hooks di React dibuat untuk memudahkan penulisan komponen.
* Hooks ini berfungsi agar functional component mampu menggunakan fitur yang sama seperti Class Component.
